# Bomb Tag 💣

Multiplayer hot potato — but the potato is a bomb.

2-6 players run around an arena. One player holds a ticking bomb. Touch another player to pass it. When the timer runs out… 💥 BOOM. Last player standing wins.

## Setup

```bash
npm install
npm start
```

Server runs on `http://localhost:3000`. Share your IP for LAN play.

## How to Play

1. Open the URL, enter your name, click **Ready**
2. Game starts when all players (min 2) are ready
3. **Bomb holder:** Chase someone and touch them to pass the bomb!
4. **Everyone else:** RUN!
5. Bomb holder gets a speed boost
6. Player holding the bomb when it explodes is eliminated
7. Last player alive wins

## Controls

- **Desktop:** WASD or Arrow Keys
- **Mobile:** Virtual joystick (touch and drag)
