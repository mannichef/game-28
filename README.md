# Bomb Tag 💣

Multiplayer hot-potato game. One player holds a ticking bomb — tag someone else before it explodes!

## Setup
```bash
npm install
node server.js
```
Open `http://localhost:3028` in multiple browser tabs.

## Controls
- **Keyboard:** WASD / Arrow keys
- **Mobile:** Virtual joystick

## Rules
- Random player starts with the bomb
- Touch another player to pass it
- Bomb explodes after 10-20 seconds
- Last player standing wins!
